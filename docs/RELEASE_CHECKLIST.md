# KANATA Terminal リリース前手動チェックリスト

対象バージョン: v0.2.0
検証環境: Windows 11 Home / Pro (クリーンインストール推奨)
前提条件: Python / Docker / WSL2 未インストール状態

## インストール検証
- [ ] `KANATA-Terminal-Setup-x.x.x.exe` をダブルクリックしてインストーラが起動する
- [ ] SmartScreen 警告が出た場合「詳細情報 → 実行」で通過できる
- [ ] インストールディレクトリを変更できる
- [ ] デスクトップ・スタートメニューにショートカットが作成される

## 起動検証
- [ ] デスクトップアイコンからアプリが 10 秒以内に起動する
- [ ] チャートが表示される（合成データでも可）
- [ ] タイトルバーに KANATA /TERMINAL が表示される

## ウォッチリスト操作
- [ ] 検索バーから AAPL を追加できる
- [ ] 検索バーから 7203 を追加できる
- [ ] 5 銘柄追加してウォッチリストに並ぶ
- [ ] 銘柄を削除できる
- [ ] 新しいウォッチリストを作成・削除できる

## チャート操作
- [ ] ローソク足チャートが描画される
- [ ] タイムフレーム 5m / 15m / 60m / 1D / 1W / 1M を切り替えられる
- [ ] 7203（日本株）でタイムフレーム切替が動作する
- [ ] パン（ドラッグ）/ ズーム（スクロール）が動作する

## オフライン時
- [ ] ネットワーク切断状態で起動すると合成データが表示される
- [ ] ウォッチリストの前回状態が維持される

## ウィンドウ操作
- [ ] 最小化 / 最大化 / 閉じるボタンが正常に動作する
- [ ] 全画面モード（メニュー → 表示 → 全画面）が動作する
- [ ] DPI 125% / 150% でも Canvas 描画が崩れない

## 永続化
- [ ] 再起動後もウォッチリストが保持される
- [ ] DB が `%APPDATA%\KANATA Terminal\kanata\kanata.db` に存在する

## アンインストール
- [ ] コントロールパネルからアンインストールできる
- [ ] アンインストール時にユーザーデータ削除の選択ダイアログが表示される
- [ ] 「残す」を選択すると `%APPDATA%\KANATA Terminal\` が残る

## ログ
- [ ] メニュー → ヘルプ → ログフォルダを開く でエクスプローラが開く
- [ ] `main.log` / `sidecar.log` が存在する
