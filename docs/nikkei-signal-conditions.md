# 日経平均上昇シグナル：条件と対応方針

## 概要

日経平均が急騰（踏み上げ相場）しやすい構造的条件6つの、データ取得方針と担当を整理する。

---

## 条件一覧

### ① 2市場合計の売り残（空売り）が 8,000億円以上

> 市場全体に「燃料」となる空売りポジションが蓄積している状態。

| 項目 | 内容 |
|---|---|
| **データ** | 信用取引現在高（買い残・売り残・2市場合計） |
| **ソース** | JPX「[信用取引現在高](https://www.jpx.co.jp/markets/statistics-equities/margin/04.html)」|
| **更新頻度** | 週次（毎週水曜 15:00） |
| **取得方法** | **手動**：JPXページから Excel をダウンロードして確認 |
| **担当** | ユーザー |
| **閾値** | 売り残合計 ≥ 8,000億円 |

---

### ② 日経レバレッジ（1570）の信用倍率が 1倍未満

> ETF において売り手が買い手を上回り、ショートカバーの起爆剤になりやすい状態。

| 項目 | 内容 |
|---|---|
| **データ** | 1570 の信用倍率（買い残 ÷ 売り残） |
| **ソース** | [株探](https://kabutan.jp/stock/credit?code=1570) / [minkabu](https://minkabu.jp/stock/1570/credit) |
| **更新頻度** | 週次（毎週金曜夕方） |
| **取得方法** | **手動**：各サイトで信用倍率を確認 |
| **担当** | ユーザー |
| **閾値** | 信用倍率 < 1.0 |

---

### ③ 日本株の優位性の拡大（NS倍率・日経÷TOPIX）

> 日本株が米国株・国内株全体に対して相対的に強い状態。ショートカバーの引き金となる。

| 項目 | 内容 |
|---|---|
| **データ** | NS倍率（^N225 ÷ ^GSPC）、日経÷TOPIX（^N225 ÷ TOPIX系）の週次時系列 |
| **ソース** | yfinance（`^N225`, `^GSPC`, TOPIX系シンボル） |
| **更新頻度** | 日次取得・週次リサンプル |
| **取得方法** | **KANATA自動取得**：マクロダッシュボードに2カード追加 |
| **担当** | KANATA バックエンド（`macro_provider.py` に `build_nikkei_sp` / `build_nikkei_topix` 追加） |
| **シグナル** | 上昇トレンド → green、下降 → yellow/red |

---

### ④ 日経平均の理論株価の大幅な上方修正（EPS）

> 企業EPSが向上し、算出される理論株価が引き上がる状態。上昇に正当性を与える。

| 項目 | 内容 |
|---|---|
| **データ** | 日経225 予想EPS、PER × EPS による理論株価レンジ（PER 18〜21倍） |
| **ソース** | [nikkei225jp.com](https://nikkei225jp.com/data/per.php) / [nikkei225fut.jp](https://nikkei225fut.jp/fundamental/nikkei) |
| **更新頻度** | 日次（決算シーズン後に大きく変動） |
| **取得方法** | **手動**：サイトで EPS・PER を確認し、理論株価レンジを目視チェック |
| **担当** | ユーザー |
| **確認ポイント** | EPSが前週比・前月比で大幅に上方修正されているか |

> **備考**：nikkei225jp.com は JS レンダリングのため API 自動取得不可。

---

### ⑤ 海外投資家の追随（買い越し）

> 海外勢が日本株の現物を大きく買い越してくる動き。踏み上げを加速させる。

| 項目 | 内容 |
|---|---|
| **データ** | 投資部門別売買状況（海外投資家の差引＝買い越し/売り越し額） |
| **ソース** | JPX「[投資部門別売買状況](https://www.jpx.co.jp/markets/statistics-equities/investor-type/index.html)」|
| **更新頻度** | 週次（毎週木曜 15:30） |
| **取得方法** | **手動**：JPXページで Excel を確認（海外投資家_差引の列） |
| **担当** | ユーザー |
| **確認ポイント** | 海外投資家_差引（`FrgnBal`）が連続して正（買い越し）になっているか |

> **備考**：J-Quants API（Lightプラン ¥1,100/月）で自動取得可能だが、現状は手動運用。

---

### ⑥ 原油価格の「ねじれ解消」に伴う上昇

> ブレント-WTI スプレッドの歪みが縮小・解消し、需給バランスが正常化する状態。

| 項目 | 内容 |
|---|---|
| **データ** | ブレント原油（`BZ=F`） − WTI原油（`CL=F`）の週次スプレッド（$/bbl） |
| **ソース** | yfinance（`BZ=F`, `CL=F`） |
| **更新頻度** | 日次取得・週次リサンプル |
| **取得方法** | **KANATA自動取得**：マクロダッシュボードに1カード追加 |
| **担当** | KANATA バックエンド（`macro_provider.py` に `build_brent_wti` 追加） |
| **シグナル** | スプレッド縮小（ねじれ解消方向）→ green、急拡大・逆転 → yellow/red |

> **備考**：通常スプレッドは $2〜$5（ブレントプレミアム）。急拡大や逆転が「ねじれ」。

---

## 担当サマリー

| 条件 | 取得担当 | 方法 |
|---|---|---|
| ① 売り残 8,000億円以上 | ユーザー（手動） | JPX Excel 週次確認 |
| ② 1570 信用倍率 < 1倍 | ユーザー（手動） | 株探/minkabu 週次確認 |
| ③ NS倍率・日経÷TOPIX | KANATA 自動 | yfinance → マクロダッシュボード |
| ④ 理論株価（EPS）上方修正 | ユーザー（手動） | nikkei225jp.com 随時確認 |
| ⑤ 海外投資家 買い越し | ユーザー（手動） | JPX Excel 週次確認 |
| ⑥ ブレント-WTI スプレッド | KANATA 自動 | yfinance → マクロダッシュボード |

---

## KANATA 実装バックログ

実装する順番を問わず、以下3つを `macro_provider.py` / `macro.py` / `MacroDashboard.tsx` に追加する。

1. **`build_nikkei_sp`** — NS倍率（^N225 ÷ ^GSPC）
2. **`build_nikkei_topix`** — 日経÷TOPIX
3. **`build_brent_wti`** — ブレント-WTI スプレッド

いずれも RSP/SPY（`build_rsp_spy`）と同じ実装パターンで追加可能。
