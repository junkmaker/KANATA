## 1. インストール方法

### 動作環境

- OS: Windows 11（64ビット）
- ネットワーク接続（株価データ取得のため必須）

### インストール手順

1. 配布された `KANATA Terminal Setup x.x.x.exe` をダブルクリックして実行します。
2. ユーザーアカウント制御（UAC）のダイアログが表示された場合は「はい」をクリックします。
3. インストールウィザードが起動します。
   - **インストール先フォルダ**を任意のパスに変更できます（デフォルトのまま推奨）。
   - 「インストール」ボタンをクリックしてインストールを開始します。
4. インストール完了後、デスクトップとスタートメニューに「KANATA Terminal」のショートカットが作成されます。
5. ショートカットをダブルクリックしてアプリを起動します。

> **注意:** 初回起動時は Python バックエンドの準備に数秒かかります。上部バーのステータスが「LIVE」（緑）になるまでお待ちください。

## 2. 基本の使い方

### 2.1 起動と初期画面

アプリを起動すると、以下の3パネル構成の画面が表示されます。

```
┌──────────────────────────────────────────────────────────┐
│  上部バー — ブランドロゴ・現在値・ステータス表示         │
├─────────────┬────────────────────────────┬───────────────┤
│  左パネル   │         メインチャート      │  右パネル     │
│  時間軸     │                             │  ウォッチリスト│
│  描画ツール │    ローソク足・指標表示     │  銘柄リスト   │
│  テクニカル │                             │  検索・フィルタ│
│  指標設定   │                             │  基本情報     │
├─────────────┴────────────────────────────┴───────────────┤
│  下部ステータスバー — 時間軸・銘柄・ツール情報           │
└──────────────────────────────────────────────────────────┘
```

上部バーのステータス表示の意味：

| 表示    | 色     | 意味                                 |
| ------- | ------ | ------------------------------------ |
| LIVE    | 緑     | バックエンド接続中・データ取得可能   |
| LOADING | 琥珀色 | データ読み込み中                     |
| ERROR   | 赤     | 接続エラー（ネットワーク確認が必要） |
| —       | グレー | 未接続                               |

---

### 2.2 ウォッチリストへの銘柄登録

**新しいウォッチリストを作成する**

1. 右パネル上部の「**+**」ボタンをクリックします。
2. 入力フィールドにリスト名を入力します（例：「日本株」「米国テック」）。
3. **Enter** キーまたは「**OK**」ボタンで確定します。

**銘柄を追加する**

1. 右パネルの「**Edit**」ボタンをクリックして編集モードに切り替えます。
2. 上部に表示された入力フォームに銘柄コードを入力します。
   - **日本株**: 4桁の数字（例：`7203`、`9984`）
   - **米国株**: 1〜6文字の英字（例：`AAPL`、`MSFT`、`NVDA`）
3. 3文字以上入力すると候補リストが表示されます。クリックまたは矢印キー + **Enter** で選択します。
4. 候補が出ない場合も、コードを直接入力して確定できます。
5. 「**Edit**」ボタンを再度クリックして編集モードを終了します。

**銘柄を削除する**

1. 「**Edit**」ボタンで編集モードに切り替えます。
2. 削除したい銘柄行の右端に表示された「**×**」ボタンをクリックします。

---

### 2.3 銘柄の選択とチャート表示

**銘柄を選択してチャートを表示する**

1. 右パネルの銘柄リストから、チャートで見たい銘柄をクリックします。
2. 選択した銘柄のデータがメインチャートに表示されます。

**複数銘柄を比較する**

- 銘柄リストの各行をクリックするたびに選択が切り替わります。
- 最大6銘柄まで同時に選択できます。
- 最初に選択した銘柄（プライマリ）のローソク足が表示され、追加銘柄は折れ線グラフで重ねて表示されます。

**プライマリ銘柄を変更する**

- 銘柄行を**ダブルクリック**すると、その銘柄がプライマリ（先頭）に設定されます。

**時間軸を切り替える**

左パネルの「**TIMEFRAME**」セクションから、見たい時間軸のボタンをクリックします。

| ボタン | 時間軸            |
| ------ | ----------------- |
| 5m     | 5分足             |
| 15m    | 15分足            |
| 60m    | 60分足（1時間足） |
| 1D     | 日足              |
| 1W     | 週足              |
| 1M     | 月足              |

**チャートの操作（パン・ズーム）**

| 操作                 | 動作                     |
| -------------------- | ------------------------ |
| チャート上でドラッグ | 時間軸の左右移動（パン） |
| マウスホイール上回し | ズームイン（拡大）       |
| マウスホイール下回し | ズームアウト（縮小）     |

## 3. 各エリア・ボタンの機能

### 3.1 上部バー（Top Bar）

| 要素               | 機能                                               |
| ------------------ | -------------------------------------------------- |
| KANATA /TERMINAL   | ブランドロゴ・バージョン表示                       |
| 銘柄コード         | 選択中のプライマリ銘柄                             |
| O / H / L / C      | 始値 / 高値 / 安値 / 終値                          |
| +xx.xx (+x.xx%)    | 前日比（価格・割合）。上昇時は緑、下落時は赤で表示 |
| ステータス（右端） | バックエンドの接続状態（LIVE / LOADING / ERROR）   |

---

### 3.2 左パネル

#### TIMEFRAME（時間軸）

`5m` / `15m` / `60m` / `1D` / `1W` / `1M` の6ボタン。クリックでチャートの時間軸を切り替えます。現在選択中の時間軸は背景色で強調されます。

---

#### DRAWING TOOLS（描画ツール）

| ボタン  | ツール名       | 操作方法                                     |
| ------- | -------------- | -------------------------------------------- |
| Pan     | パン（移動）   | ドラッグでチャートをスクロール（デフォルト） |
| Trend   | トレンドライン | ドラッグで始点〜終点を結ぶ直線を引く         |
| Hline   | 水平線         | クリックした価格に水平の破線を引く           |
| Vline   | 垂直線         | クリックした時点に垂直の破線を引く           |
| Rect    | 矩形           | ドラッグで半透明の矩形エリアを描く           |
| Ellipse | 楕円           | ドラッグで楕円を描く                         |
| Text    | テキスト       | クリック後、テキストを入力して注釈を追加する |
| CLEAR   | 全削除         | すべての描画オブジェクトを一括削除する       |

> ツールをクリックして有効化すると、チャート上の操作がそのツールの動作に切り替わります。もう一度クリックすると Pan（移動）モードに戻ります。

---

#### TECHNICAL ANALYSIS（テクニカル指標）

各指標左のドットをクリックしてオン/オフを切り替えます。点灯しているものが有効です。詳細は「[4. テクニカル指標](#4-テクニカル指標)」を参照してください。

---

#### FUNDAMENTALS（ファンダメンタルズ）

- **Show fundamentals pane** トグル: チャート下部にファンダメンタルズパネルを表示/非表示します。
- 有効時は ROE / ROIC / PER の表示チェックボックスが現れます。

---

#### VOLUME（出来高）

- **Show volume** トグル: チャート下部に出来高バーを表示/非表示します。

---

### 3.3 メインチャート

**チャート構成（上から順に表示）**

```
メインチャート（ローソク足 + 移動平均線 + オーバーレイ指標）
Volume バー        [Show volume が有効な場合]
Stochastics        [Stoch が有効な場合]
MACD               [MACD が有効な場合]
RSI                [RSI が有効な場合]
Fundamentals       [Show fundamentals が有効な場合]
```

**チャート左上の情報（Legend）**

カーソル位置または最新バーの OHLC 値と各指標の現在値が表示されます。

```
銘柄コード
O: 始値  H: 高値
L: 安値  C: 終値（色付き）  変動幅
V: 出来高
[各指標の現在値]
```

**カーソル操作**

- チャート上にマウスを乗せると十字カーソル（クロスヘア）が表示されます。
- 縦軸右端に価格、横軸下端に日時がリアルタイムで表示されます。

---

### 3.4 右パネル

#### ウォッチリスト操作ボタン

| ボタン/要素                | 機能                                                                            |
| -------------------------- | ------------------------------------------------------------------------------- |
| ドロップダウン（リスト名） | 複数のウォッチリストを切り替える。各リストの銘柄数が表示される                  |
| Edit                       | 編集モードのオン/オフ切り替え。有効時に銘柄追加フォームと削除ボタンが表示される |
| +                          | 新規ウォッチリストを作成する                                                    |

**編集モード中の追加操作**

| 操作          | 機能                                                  |
| ------------- | ----------------------------------------------------- |
| Rename ボタン | ウォッチリストの名前を変更する                        |
| Delete ボタン | 現在のウォッチリストを削除する（最後の1件は削除不可） |

#### 検索・フィルター

- **検索ボックス**: 銘柄コード・銘柄名・セクターで絞り込み検索ができます。
- **[ALL] [JP] [US]** タブ: 市場（全市場 / 日本株 / 米国株）でフィルタリングします。

#### 銘柄リスト

各行の構成：

```
[チェック] コード  市場  │  スパークライン  │  価格
          銘柄名        │  （直近推移）    │  変動率
```

| 要素           | 説明                                                     |
| -------------- | -------------------------------------------------------- |
| チェックマーク | 選択状態。プライマリは「●」、2番目以降は「2」「3」と表示 |
| スパークライン | 直近の価格推移を小さなグラフで表示。上昇は緑、下落は赤   |
| 価格 / 変動率  | 最新の株価と前日比（%）                                  |

**銘柄行の操作**

| 操作                     | 動作                                     |
| ------------------------ | ---------------------------------------- |
| クリック                 | 選択/解除のトグル（最大6銘柄まで選択可） |
| ダブルクリック           | プライマリ銘柄に設定する                 |
| × ボタン（編集モード時） | リストから銘柄を削除する                 |

#### FUNDAMENTALS（基本情報）

プライマリ銘柄のファンダメンタル情報を表示します：

| 指標     | 内容                 |
| -------- | -------------------- |
| ROE      | 自己資本利益率（%）  |
| ROIC     | 投下資本利益率（%）  |
| PER      | 株価収益率（倍）     |
| PBR      | 株価純資産倍率（倍） |
| DIV      | 配当利回り（%）      |
| MCAP     | 時価総額             |
| セクター | 業種                 |

---

### 3.5 下部ステータスバー

チャートの現在の状態をテキストで常時表示します。

```
TF 1D  |  SYM AAPL · MSFT  |  TOOL PAN  |  DRAW 3
Drag to pan · scroll to zoom · double-click row to set primary
```

| 情報 | 説明                             |
| ---- | -------------------------------- |
| TF   | 現在の時間軸                     |
| SYM  | 選択中の銘柄（「·」区切り）      |
| TOOL | 現在の描画ツール                 |
| DRAW | チャート上の描画オブジェクトの数 |

## 4. テクニカル指標

左パネルの「TECHNICAL ANALYSIS」セクションで各指標をオン/オフします。

### Moving Averages（移動平均線）

メインチャートに重ねて表示されます。

| 指標   | 説明                         | 表示色   |
| ------ | ---------------------------- | -------- |
| SMA 5  | 5期間の単純移動平均          | 琥珀色   |
| SMA 25 | 25期間の単純移動平均         | 水色     |
| SMA 75 | 75期間の単純移動平均         | マゼンタ |
| EMA 20 | 20期間の指数移動平均（破線） | ライム   |

### Overlays（オーバーレイ）

メインチャートに重ねて表示されます。

| 指標                 | 説明                                     |
| -------------------- | ---------------------------------------- |
| Bollinger Bands 20,2 | 期間20・標準偏差2のボリンジャーバンド    |
| Parabolic SAR        | トレンド転換シグナルをドットで表示       |
| Ichimoku             | 一目均衡表（転換線・基準線・遅行線・雲） |

### Oscillators（オシレーター）

メインチャートの下のサブペインに表示されます。

| 指標               | 説明                                     | カスタムパラメータ             |
| ------------------ | ---------------------------------------- | ------------------------------ |
| Stochastics 14,3,3 | ストキャスティクス（%K・%D）             | なし                           |
| MACD               | MACDライン・シグナルライン・ヒストグラム | Fast / Slow / Signal 期間      |
| RSI                | 相対力指数（0〜100）                     | Period / Overbought / Oversold |

**MACD パラメータの変更（デフォルト: Fast=12, Slow=26, Signal=9）**

左パネルの各入力欄に数値を直接入力して変更します。設定可能範囲：Fast/Slow は 1〜200、Signal は 1〜50。

**RSI パラメータの変更（デフォルト: Period=14, OB=70, OS=30）**

設定可能範囲：Period は 2〜200、Overbought は 50〜99、Oversold は 1〜49。

## 5. 描画ツール

### 基本的な使い方

1. 左パネルの「DRAWING TOOLS」から使いたいツールをクリックして選択します。
2. チャート上で操作します（下表参照）。
3. 描画が完了したら「Pan」ボタンをクリックして通常の移動モードに戻ります。

### 各ツールの操作

| ツール                  | チャート上での操作           | 結果                         |
| ----------------------- | ---------------------------- | ---------------------------- |
| Trend（トレンドライン） | ドラッグ（始点→終点）        | 2点間を結ぶ直線              |
| Hline（水平線）         | 任意の場所をクリック         | クリックした価格に水平の破線 |
| Vline（垂直線）         | 任意の場所をクリック         | クリックした時点に垂直の破線 |
| Rect（矩形）            | ドラッグ                     | 半透明の矩形エリア           |
| Ellipse（楕円）         | ドラッグ                     | 楕円形                       |
| Text（テキスト）        | クリック後、ダイアログに入力 | テキスト注釈                 |

### 描画の削除

**CLEAR ボタン**（左パネル下部）でチャート上のすべての描画をまとめて削除します。

## 6. アンインストール方法

### 方法1: Windowsの設定から削除

1. スタートメニューを開き、「設定」→「アプリ」→「インストールされているアプリ」を開きます。
2. 一覧から「KANATA Terminal」を探します。
3. 右端の「…」メニューをクリックし、「アンインストール」を選択します。
4. 確認ダイアログで「アンインストール」をクリックします。

### 方法2: コントロールパネルから削除

1. スタートメニューで「コントロールパネル」を開きます。
2. 「プログラム」→「プログラムのアンインストール」を開きます。
3. 「KANATA Terminal」を右クリックし、「アンインストール」を選択します。

### アンインストール後のデータについて

アンインストールしてもユーザーデータ（ウォッチリスト・設定など）は以下の場所に残る場合があります：

```
C:\Users\<ユーザー名>\AppData\Roaming\kanata\
```

データを完全に削除する場合は、上記フォルダを手動で削除してください。

> **注意:** フォルダを削除すると、登録したウォッチリストや設定がすべて失われます。必要な場合は削除前にバックアップしてください。
